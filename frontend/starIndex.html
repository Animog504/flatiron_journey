How to Play

Press key right to walk right.

Press key left to walk left.

You will be stopped by a monster for 5 times.

The monster is a symbol of code challenge for each module.

You can interact with the monster for a short conversation and a possible code challenge.

About this game

This game is a module 3 final project made by Zoe, Jing, and Jack at Flatiron School.

It illustrates our coding journey at Flatiron's coding bootcamp.

Please see the project's github page for details.
